# Agromash Elit project

###  1. Clone repository
```
git clone https://github.com/evilenemy/agromashelit.git
cd agromashelit
```

### 2. Install yarn
```
npm i -g yarn
```

### Start frontend of the project
```
cd client
yarn
yarn dev
```

### Start backend of the project
```
cd server
yarn
yarn dev
```

### Start admin dashboard of the project
```
cd admin
yarn
yarn dev
```
